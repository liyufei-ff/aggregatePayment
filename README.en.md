# 基于微服务开发的聚合支付平台

#### Description
一个基于微服务架构的聚合支付的平台，主要的做法就是线上聚合收银台，线下C2B、一码多
付、线下B2C商家扫码。平台以SaaS服务形式提供给商户订单管理、门店管理、支付管理等基础服务，构
建起一个移动支付体系。
